# Documentation

This directory is the source of truth for project decisions and methodology.
Planned documents include:

- project charter and scope
- source-data manifest and data dictionary
- metric definitions and assumptions register
- power-user, churn, and catalog-exhaustion methodology
- candidate-score and portfolio-constraint specification
- offline evaluation and sensitivity analysis
- experiment plan, limitations, and ethical considerations
- résumé bullets, LinkedIn draft, and interview talking points

Material analytical choices should also receive a short record in `decisions/`.
